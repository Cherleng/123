## 取出个位数、十位数、百位数...的方法 | 以PTA程序设计基础题 - 倒序输出数字为例

{% include alert.html text="本文作者：1900 原稿发表于2021-11-28 本文受版权保护，转载请联系编辑部." %}


示例：

123


输出：


321




完整代码如下：


```c


#include <stdio.h>

int main(){
    int a,b,c,n,n2;
    scanf("%d",&n);
    a=n%10;
    b=n/10%10;
    c=n/100;
    n2=a*100+b*10+c;
    printf("%d",n2);
    
    return 0;
}

```
